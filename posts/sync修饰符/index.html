sync修饰符

vue sync修饰符的功能是：在一个包含 title prop 的假设的组件中，我们可以用以下方法表达对其赋新值的意图：

this.$emit('update:title', newTitle)

然后父组件可以监听那个事件并根据需要更新一个本地的数据属性。

例子：一个+1按钮
父组件App.vue：

<template>
  <div id="app">
    {{val}}
    <Child :num.sync="val"/>
  </div>
</template>

<script>
import Child from "./Child.vue";

export default {
  data() {
    return { val: 0 };
  },
  components: { Child }
};
</script>

子组件Child.vue：
<template>
  <button @click="$emit('update:num',num++)">+1</button>
</template>

<script>
export default {
  props: ["num"]
};
</script>
