computed 计算属性： 从其他属性计算而来的属性变成一个属性,调用时不需要加括号。 会根据依赖数据发生的改变，进行计算，显示新的结果，这个结果支持缓存，所以只有当依赖的数据改变时，才会重新计算. 不支持异步，当computed内有异步操作时无效，无法监听数据的变化. 计算属性将被混入到 Vue 实例中。所有 getter 和 setter 的 this 上下文自动地绑定为 Vue 实例。 var vm = new Vue({ data: { a: 1 }, computed: { // 仅读取 aDouble: function () { return this.a * 2 }, // 读取和设置 aPlus: { get: function () { return this.a + 1 }, set: function (v) { this.a = v - 1 } } } }) vm.
